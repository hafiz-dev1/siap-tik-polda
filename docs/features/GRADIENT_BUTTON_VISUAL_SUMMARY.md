## 🎭 Visual Effects

### Normal State
```
┌──────────────────┐
│   Button Text    │  ← gradient colors
│  shadow-md       │  ← medium shadow
└──────────────────┘
```

### Hover State
```
┌──────────────────┐
│   Button Text    │  ← darker gradient
│  shadow-lg ▓▓    │  ← larger shadow
└──────────────────┘
    ↑ smooth 200ms transition
```

### Disabled State
```
┌──────────────────┐
│   Button Text    │  ← gray gradient
│  no shadow       │  ← cursor-not-allowed
└──────────────────┘
```
